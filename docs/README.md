## **■ キャリアプロフィール**

私は、ハードウェア開発から Web アプリケーション開発まで幅広い経験を持つエンジニアです。直近では、Fabeee 社にて約 2 年間、主に Python や TypeScript を使用した Web 開発プロジェクトに携わってきました。

常に新しい技術やベストプラクティスを学び、プロジェクトに適用することを心がけています。また、効率的な開発プロセスや保守性の高いコードの作成に注力し、チームの生産性向上に貢献してきました。

特定の技術だけでなく、広範囲に汎用的に技術を深めたいと思って、業務に取り組んでいます。

## ■ 活かせる経験・知識・技術

- フルスタック Web 開発:
  - Django や FastAPI を使用したバックエンド開発、Next.js や Vue.js を用いたフロントエンド開発の経験があります。
- データ処理と可視化:

  - データ可視化作業を自動化して、業務を効率化しました。
- パフォーマンス最適化:
  - C++や Cython を用いて既存システムの処理速度を向上させた経験があります。

## ■ 受託開発「空気清浄機 組み込み Linux 開発」(2020 年 8 月～ 2020 年 12 月)

RaspberryPi4 により、スマートフォン・本体の各デバイス間の通信制御機能等の組み込み Linux 開発業務

### 〇担当した開発機能 (実装言語・使用ライブラリ)

- ログ機能(センサ値ログ・操作ログ)(Python)
- スレッド(C, pthread)
- SPI 通信機能(C)
- BLE 通信機能(C++,BlueZ)
- USB マウント監視(Python)

### 〇業務内容

キャリア初の組み込み Linux 開発グループのリーダーとしてプロジェクトに参画した。

Android チームと連携し、BLE 通信の設計・開発する。また、空気清浄機本体の制御側のエンジニアと連携し、SPI 通信の設計・開発を行う。
クライアントからの要求を UML 図(シーケンス図・状態遷移図)に起こし、認識違いを防ぎつつ開発を進めた。

### 〇当プロジェクトからの経験

- 通信設計書・試験仕様書等のドキュメント作成スキル
- 組み込み Linux 開発に必要な技術知識(LPIC1 相当)
  - Systemd,デバイス監視 ,アプリ自動起動, プロセス監視, 環境構築スクリプト(基本的な Linux コマンド)
- UML モデリング(状態遷移図・シーケンス図・ユースケース図・アクティビティ図)
